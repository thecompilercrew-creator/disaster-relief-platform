## Requesting Help

1. Go to the **Requests** tab
2. Click **Add Request**
3. Fill in:
   - What help you need
   - Address/location
   - Urgency level
   - Contact details
4. Submit the request

Your request will appear on the map.

---

## Volunteering to Help

1. Open the **Requests** tab
2. Browse existing requests
3. Click **Volunteer** on a request
4. Contact details will be revealed after volunteering

---

## Offering Resources

1. Go to the **Resources** tab
2. Click **Add Resource**
3. Enter:
   - Type of resource
   - Quantity
   - Address
   - Contact info

This helps others locate available aid quickly.
